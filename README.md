# Web Desa Cibatu (Modern Interface)

Website resmi Desa Cibatu yang dibangun menggunakan teknologi web modern untuk memberikan layanan informasi dan transparansi publik yang responsif, cepat, dan aman.

## 🚀 Teknologi
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Lottie React & Framer Motion

## ✨ Fitur Utama
- **Profil Desa:** Informasi sejarah, visi, misi, dan struktur organisasi.
- **Layanan Surat Online:** Integrasi pengajuan surat bagi warga.
- **Transparansi Dana:** Grafik dan data real-time anggaran desa.
- **Berita & Pengumuman:** Update terbaru seputar kegiatan desa.
- **Peta Wilayah:** Visualisasi batas dan titik penting desa.

## 🛠️ Cara Menjalankan (Development)
1. Clone repository:
   ```bash
   git clone https://github.com/SunnFlower47/web-desa.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Jalankan server lokal:
   ```bash
   npm run dev
   ```
